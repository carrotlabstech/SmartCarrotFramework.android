Notifiable

interface Notifiable

An interface to allow objects to subscribe to NotificationCenter.NotificationName events. An object which implements this interface can be passed as a parameter to NotificationCenter.addObserver and NotificationCenter.removeObserver methods in order to subscribe / unsubscribe from NotificationCenter.NotificationName events.
