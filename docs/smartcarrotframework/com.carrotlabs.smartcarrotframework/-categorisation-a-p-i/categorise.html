fun categorise(description: String, amount: BigDecimal, isPersonal: CategorisationType = CategorisationType.personal): Int

Categorises a transaction

Parameters

description - transaction description

amount - transaction amount

isPersonal - if personalisation should be applied. Set it to CategorisationType.personal if you want to apply user personalisation settings, and to CategorisationType.nonPersonal if you want to use generalised model. Default value is CategorisationType.personal`.

Exceptions

NotAuthorisedCarrotContextError - if CategorizationContext has not been authorized

FailedToLoadModelCarrotContextError - if the model didn't load

Return
